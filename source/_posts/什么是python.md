python 是一门动态语言， 定义变量时不需要声名变量的类型

python 是一种解释型语言，在运行代码前不需要提前编译

python中一切皆对象，他有对象模块，线程，异常处理和自动内存管理等，程序员可以专注与算法和数据结构，而不必处理底层细节
